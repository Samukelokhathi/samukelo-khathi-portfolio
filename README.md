
# Samukelo Khathi Portfolio (HTML)

This is a basic portfolio website built only with **HTML**.  
It introduces me as a beginner developer learning web development at **CodeTribe Academy**.

## Sections
- **Header**: Logo, navigation links (Home, About Me, Contact Us), and main image.  
- **About Me**: Heading, short introduction, and profile image.  
- **Footer**: Address, navigation links, embedded Google Map, and social media icons (LinkedIn, GitHub, Facebook, WhatsApp).  

## Current State
- Pure HTML structure with text, images, links, and an embedded map.  
- No CSS or JavaScript yet
- Preview => https://samukelo-khathi-portfolio-codetribe.vercel.app/
